# PN Junction Diode Virtual Lab

This project simulates the V-I characteristics of a PN Junction Diode using Streamlit.

## Features
- Forward and reverse bias simulation
- Adjustable temperature, ideality factor, and saturation current
- Real-time graph plotting
- Cut-in voltage estimation
- Download simulation data

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
